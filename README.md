## Table of Contents


## Install

[npm][]:

```sh
npm install zw-rec
```

[yarn][]:

```sh
yarn add zw-rec
```


## Usage

```js
const ZwRec = require('zw-rec');

const zwRec = new ZwRec();

console.log(zwRec.renderName());
// script
```
